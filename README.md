# git-glob-copy
> Glob-aware two-way copying for git

## Usage
```shell
ggcp ./packages/*/bundle/**/*.js git://github.com/qiwi/git-glob-copy/master/artifacts/
```

### Options
| Flag | Description    | Default       |
|------|----------------|---------------|
| `-m` | Commit message | `chore: sync` |
|
